# GlassGradients 1.1.0

Install the maintained line:

```bash
npm i glassgradients@latest
```

## Highlights

- separated `surface`, `gradient`, and `frost` modes while keeping older API usage working
- expanded recipes, semantic contracts, contrast-safe tuning, and static performance profiles
- official component primitives for panels, buttons, navbars, dialogs, command palettes, tables, inspectors, terminals, workspaces, forms, and more
- framework adapters for React, Vue, Solid, Preact, Svelte, Vanilla, Tailwind, UnoCSS, Next, Nuxt, and Astro
- SSR helpers for theme bootstrapping and framework-safe style props
- generated GitHub Lab with charts, metrics, component previews, adapter matrix, runtime lab, and test matrix
